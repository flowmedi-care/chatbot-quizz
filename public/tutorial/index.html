Como usar o quiz

Guia completo: WhatsApp (bot) e página do quiz no navegador.

Conteúdo do tutorial

Tudo funciona em duas frentes: o WhatsApp (bot que cria questões, recebe respostas e publica no grupo) e a página do quiz (ranking e questões para responder no navegador). Leia na ordem ou use o índice abaixo.

1. Iniciar o chat com o bot 2. Criar uma questão 3. Responder pelo WhatsApp 4. Usar a página do quiz 5. Todos os comandos 6. No grupo do WhatsApp

1. Iniciar o chat com o bot (privado)

O bot só interpreta comandos de quiz quando você ativa o modo no seu WhatsApp pessoal. Assim suas mensagens normais não viram comando por engano.

  1. Abra uma conversa no privado com o número do bot (o mesmo que publica no grupo).
  2. Envie /quiz. O bot confirma e liga o modo; a partir daí ele entende criação de questões, respostas e comandos de ajuda neste chat.
  3. Para ver o guia de novo a qualquer momento: /ajuda ou a palavra guia.
  4. Para voltar ao chat normal (sem o bot interpretar tudo): envie /quizoff. Com o modo desligado, no privado só /quiz é respondido — útil quando você só quer conversar.
Dica: No grupo, qualquer pessoa pode usar /ajuda para ver o mesmo guia no próprio grupo.

2. Criar uma questão (privado, modo /quiz ligado)

O bot guia você passo a passo. Responda cada etapa no privado.

  1. Envie a frase nova questao (sem aspas).
  2. Tipo de questão: responda 1 para múltipla escolha (A–E) ou 2 para certo ou errado.
  3. Enunciado: pode ser texto, imagem, print ou PDF — o que o bot pedir, envie na conversa.
  4. Gabarito:
    • Múltipla escolha: uma letra A, B, C, D ou E.
    • Certo ou errado: C (certo) ou E (errado).
  5. Comentário do autor (opcional): texto, imagem ou PDF com a resolução; ou envie pular para não incluir.

Depois de salvar, a questão ganha um número curto (ID). Esse número aparece no grupo e nas mensagens do bot — use-o para responder e para comandos como /gabarito.

3. Responder pelo WhatsApp (privado)

O formato é sempre: letra + espaço + número da questão (o ID que o bot mostrou).

  • Múltipla escolha: a 182, b 182, … e 182 (exemplo com questão 182).
  • Certo/errado: c 182 (certo) ou e 182 (errado).

Cada pessoa tem uma resposta por questão. Se tentar de novo, o bot pergunta se você quer alterar a última resposta: responda sim para trocar ou nao para manter.

4. Usar a página do quiz no navegador

  1. Ranking: na página inicial, na primeira seção, você vê o pódio e a tabela com posição, nome e quantidade de acertos (dados do mesmo sistema do bot).
  2. Questões: na grade, cada cartão é uma questão publicada. Use os filtros por pessoa e por acerto/erro para encontrar o que precisa. Clique no cartão para abrir o enunciado.
  3. O botão Relatório gera um ZIP com Markdown e mídias das questões e das respostas do WhatsApp — útil para mandar a uma IA e montar revisões (por exemplo, só as questões erradas).
  4. Dentro da janela da questão, escolha a alternativa (A–E ou certo/errado). O site diz se você acertou e permite abrir o gabarito oficial e o comentário do autor quando existir.
Respostas pelo site são só para estudo na hora; o ranking e o relatório usam as respostas registradas pelo bot no WhatsApp.

5. Referência de comandos e palavras-chave

Privado (com modo quiz ligado), salvo onde indicado.

O que enviar O que acontece
/quiz Liga o modo quiz para o seu contato e mostra o guia.
/quizoff Desliga o modo; mensagens normais deixam de ser comandos.
/ajuda ou guia Mostra o guia completo de novo.
nova questao Inicia o assistente para criar uma questão.
a 182 … e 182 Responde a questão 182 com a letra indicada (troque pelo número real).
/questao 182 ou questao 182 Repete o enunciado salvo (texto/mídia). Funciona também no grupo.
/gabarito 182 Mostra resultado completo: gabarito, estatísticas, quem acertou/errou, comentário. Grupo ou privado.
ranking Lista ordenada por acertos. No privado usa o grupo configurado no servidor.

6. No grupo do WhatsApp

  • /ajuda — mesmo guia, para todos verem.
  • ranking — ranking daquele grupo.
  • /gabarito + número — resultado da questão.
  • /questao + número — repetir o enunciado.

Com o modo quiz desligado no privado, no grupo esses comandos continuam disponíveis; só o fluxo de criar questão e responder por comando costuma ser no privado com /quiz ativo.
