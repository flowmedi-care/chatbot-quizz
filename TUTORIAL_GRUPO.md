# Tutorial — Bot de questões (para enviar no grupo)

Copie e cole no WhatsApp. Você pode editar o nome do grupo ou do bot.

---

Olá, pessoal.

Este grupo usa um **bot de questões** para treinar com alternativas e ver ranking.

## No grupo

| O que fazer | Como |
|-------------|------|
| Ver **gabarito e resultado** de uma questão (distribuição, quem acertou, comentário do autor) | Envie: `/gabarito` + espaço + **número da questão** (ex.: `/gabarito 5`) |
| **Repetir o enunciado** (texto/imagem/PDF salvos) | Envie: `/questao` + espaço + número (ex.: `/questao 5`) |
| Ver **ranking** de quem mais acertou | Envie: `ranking` |
| Ver o **guia completo** de uso no grupo ou no privado | Envie: `/ajuda` |

---

## No privado com o bot (importante no número pessoal)

Para o bot **não interpretar** suas conversas normais como comando:

1. Envie **`/quiz`** uma vez — o modo quiz **liga** para você (fica salvo) e o bot manda o **guia**.
2. Use os comandos (`nova questao`, responder com `c 5`, etc.).
3. Para voltar ao chat normal: **`/quizoff`**
4. Para ler o guia de novo: **`/ajuda`** ou **`guia`**

---

## Responder uma questão

Sempre **no privado do bot**, depois que a questão foi publicada no grupo:

| Tipo | Exemplo |
|------|---------|
| Múltipla escolha | `a 5` ou `c 5` (letra + espaço + número da questão) |
| Certo ou errado | `c 5` (certo) ou `e 5` (errado) |

Se você já respondeu e mandar outra letra, o bot pergunta se quer **alterar** — responda `sim` ou `nao`.

---

## Criar uma questão (quem for mandar enunciado)

No **privado**, com modo quiz ativo (`/quiz`):

1. `nova questao`
2. Escolha `1` (múltipla escolha) ou `2` (certo ou errado)
3. Envie o **enunciado** (texto, imagem ou PDF)
4. Envie o **gabarito** (A–E ou C/E)
5. Comentário opcional ou **`pular`**

A questão é publicada automaticamente no grupo.

---

## Resumo rápido

- **Grupo:** `/gabarito ID` · `ranking` · **`/ajuda`**
- **Privado:** **`/quiz`** para ligar · **`/quizoff`** para sair · **`/ajuda`** para o guia

Qualquer dúvida, mandem **`/ajuda`** aqui no grupo ou no privado com o bot.
